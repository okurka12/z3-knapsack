Solve Knapsack problem using z3

This program uses z3's python API
https://github.com/Z3Prover/z3?tab=readme-ov-file#python

Author: Vit Pavlik (xpavli0a)

Date: 2024-06-06

Assignment for the IAM course, FIT VUT

Developed with:
 - Debian 12.5
 - Python 3.12.3
 - z3 PyPi package version 4.13.0.0

How to run:
 - encode the knapsack problem into knapsack.py (OBJECTS, KNAPSACK_CAPACITY)
 - on linux, run `python3 knapsack.py | grep o | tr "[" " " | sort -V`
